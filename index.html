PROJECTS

Socket Programming and Client-Server Architecture

This project explains the use of sockets for communicating over the Internet. Ultimately an Overlay Network is formed and thus network communication can be experienced in the overlay. The objective is to learn to handle multiple clients with a Connectionless Iterative Server using UDP and Connection-oriented Concurrent Server using TCP for communication. Overlay Network is created using PlanetLab. PlanetLab is an open, globally distributed platform used for developing, deploying and accessing planetary-scale network services.

Simulation based Evaluation of Network Interface Card

The Network Interface Card (NIC) allows computers to communicate over a local-area network. A typical NIC is both a physical layer and data link layer device and provides access to the transmission medium, and supports a MAC protocol and an addressing scheme. This project helps in evaluating the performance of a NIC using an event-driven simulator.

Searching Contents in a Distributed Application-Layer Network

This project develops a solution to search for contents in a distributed system. A simple overlay solution is developed which allows a set of nodes to share resources among each other. A set of nodes is connected via some overlay topology. Each node has a set of files that it is willing to share with other nodes. A node in the system which is looking for a particular file issues a query to identify a node that contains the specific mentioned file. Once the node is identified, it can then be exchanged between the two nodes. The main goal is to generate network topology and design searching algorithms to locate interesting items.

Implementation of Caching in Unstructured Peer to Peer Networks

Implements a caching mechanism in an overlay unstructured Peer to Peer network that allows a set of nodes to share as well as cache pointers to resources among each other. A set of nodes is connected via an unstructured P2P overlay topology. Each node has a set of files that it is willing to share with other nodes and each node has cache of size N which holds IP address pairs. This way the search query does not have to propogate all the way to the node with the contents. Thus, this project helps to develop an efficient solution to search for conents in a distributed system.
